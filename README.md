# Online Retail Data Cleaning and Quality Analysis

## Overview
This project performs a data quality assessment and cleaning pipeline on the Online Retail dataset from the UCI Machine Learning Repository. The analysis focuses on identifying common data issues in transactional datasets and preparing the data for reliable downstream analysis.

## Methods
- Missing value analysis
- Duplicate detection
- Detection of invalid quantities and prices
- Identification of cancelled transactions
- Text standardization for product descriptions and country names
- Data consistency checks across product and customer attributes

## Data
UCI Machine Learning Repository — Online Retail Dataset  
~540,000 UK retail transactions from December 2010 to December 2011.

## Tools
Python, pandas, NumPy, fuzzywuzzy
